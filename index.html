Projet - 2ème année

RFTG

Le projet RFTG (Raised From The Graveyard) est une solution complète de gestion de locations de DVD, composée de :

• PEACH : Base de données centralisée (films, clients, réservations).
• TOAD : API REST pour la communication entre les services.
• MARIO : Application web (Laravel) pour les gérants : gestion du catalogue, du stock et suivi des locations.
• LUIGI : Application Android pour les clients : consultation, réservation et suivi des locations.

Le projet combine développement web, mobile, API et base de données pour répondre aux besoins des gérants et offrir une expérience fluide aux utilisateurs.
